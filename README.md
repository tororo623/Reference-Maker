# 参考文献フォーマットメーカー (A Reference Maker)
ブラウザ上でショートカットキーを押すだけで、開いているWebページの情報を学術レポートや論文で使える参考文献フォーマットに整え、クリップボードにコピーするブラウザ拡張機能です。対応ブラウザは「Google Chrome」「Microsoft Edge」になります。

# 主な機能
ショートカットキー一発でコピー: デフォルトでは Ctrl + Shift + Z で動作します。

著者名・題名の自動推測: ページのメタデータやタイトルから、著者名やサイト名、記事タイトルを自動で抽出して入力欄にセットします。

柔軟な編集: コピー前にポップアップで著者名や題名を微調整できます。

フォーマット: 「著者名, 『題名』, URL, 閲覧日.」で出力します。

# インストール方法
1.このリポジトリをZIP形式でダウンロードし、ZIPファイルを右クリックし「すべて展開」を選択し、解凍します。

2.この時、「

2.Google Chrome または Microsoft Edge の拡張機能管理画面（`chrome://extensions/`または`edge://extensions/`）を開きます。

3.「デベロッパー モード」をオンにします。

4.「パッケージ化されていない拡張機能を読み込む」を選択し、解凍したフォルダを指定します。

5.読み込みをしたら終了！

# 使い方
1.参考文献として引用したいWebページを開きます。

2.`Ctrl + Shift + Z`を押します。

3.ポップアップが表示されるので、著者名と題名を確認・修正して「OK」を押します。

4.クリップボードにコピーされたテキストを、レポートや論文の末尾に貼り付けてください。
